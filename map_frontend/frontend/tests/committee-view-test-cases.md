## 九、会议生命周期完整流程

### TC-9.1 完整会议流程（E2E）

```
秘书创建会议 → 秘书催缴 → 委员填报 → 秘书代填 → 
主任委员查看统计 → 主任委员提交决议 → 秘书生成纪要 → 秘书下发纪要 → 会议归档
```

| 步骤 | 角色 | 操作 | 预期 |
|------|------|------|------|
| 1 | 秘书 | 一键新增会议 | 会议创建，状态「进行中」 |
| 2 | 秘书 | 催缴未提交委员 | 催缴标记显示 |
| 3 | 委员A | 填写观点（评分+核心观点）→ 提交 | 提交成功 |
| 4 | 秘书 | 代填委员B观点 → 提交 | 提交成功 |
| 5 | 主任委员 | Tab 03 查看委员会统计 | 显示已提交委员的共识分析 |
| 6 | 主任委员 | Tab 04 填写决策 → 提交决议 | 决议提交成功，会议状态→已结束 |
| 7 | 秘书 | Tab 05 点击 AI 生成纪要 | 纪要内容生成 |
| 8 | 秘书 | 点击下发纪要 | 纪要下发，会议状态→已归档 |
| 9 | 所有角色 | 返回门户查看 | 会议显示为「已归档」 |
