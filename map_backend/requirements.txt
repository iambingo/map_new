# MAP Backend — 完整依赖清单（由代码扫描生成 + 显式底层驱动）
# Python 3.10+  |  pip install -r requirements.txt
#
# 【重要】依赖必须装进「运行 uvicorn 的同一个解释器」里（推荐项目 venv）：
#   cd backend
#   .\venv\Scripts\pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
# 若只用全局/Anaconda 的 pip 安装，而 uvicorn 用 backend\venv，会出现 ModuleNotFoundError。
#
# 扫描范围：backend/**/*.py
# 第三方包映射：
#   fastapi, starlette          → Web 框架
#   uvicorn[standard]           → ASGI 服务器（含 uvloop / httptools / watchfiles）
#   sqlalchemy                  → ORM + 异步引擎（隐式依赖 greenlet）
#   greenlet                    → SQLAlchemy asyncio 必需（显式列出避免环境残缺）
#   aiomysql                    → mysql+aiomysql 异步驱动
#   pymysql                     → PyMySQL：aiomysql 底层同步协议栈 / 同步工具脚本
#   pydantic, pydantic-settings → 校验与配置
#   python-jose[cryptography]   → JWT（jose 包名）
#   cryptography                → JWT 非对称算法 / TLS（python-jose 强烈建议）
#   passlib[bcrypt], bcrypt     → 密码哈希
#   httpx, httpcore, certifi    → 异步 HTTP（httpcore/certifi 为 httpx 链，显式防残缺）
#   tenacity                    → Adapter 重试
#   redis[hiredis], hiredis    → 异步 Redis + C 解析加速
#   alembic                     → 数据库迁移（与 SQLAlchemy 配套）
#   pytest, pytest-asyncio      → 测试
# =============================================================================
